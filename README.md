🌍 Eco-Balance
An interactive climate sandbox and ecosystem simulator where you control the environment in real time. Tweak rainfall, temperature, sunlight, and air quality to keep a fragile food chain alive. Track live resource charts and survive sudden disasters like wildfires or blizzards before a single extinction crashes the whole ecosystem.

🎮 Features
Real-Time Climate Sliders: Directly adjust rainfall, temperature, sunlight, decomposition rates, and air quality.

Dynamic Food Chain: Autonomous behavior-driven entities including Trees, Insects, Rabbits, Wolves, Fungi, and Parasites.

Context-Aware Disasters: Sudden environmental events (Droughts, Floods, Wildfires, Blizzards) trigger dynamically based on your current climate settings.

Live Analytics Dashboard: Real-time multi-species population graphs, carrying capacity indicators, and ecosystem health gauges.

🛠️ Tech Stack
Frontend: Clean, responsive semantic HTML5 and custom CSS layouts.

Simulation Engine: Vanilla JavaScript (ES6+) utilizing state-machines for entity behaviors.

Graphics: HTML5 Canvas for smooth, low-overhead micro-world rendering.

🔧 Setup & Installation
Because this project runs entirely client-side without heavy external frameworks, running it locally is incredibly straightforward:

Clone or download this repository.

Ensure index.html, style.css, and sim.js are all saved inside the same project folder.

Open index.html in any modern web browser to launch the simulation immediately.
